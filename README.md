# 📊 Order-to-Cash (O2C) Analytics Dashboard

🚀 Project built for Analytics / Consulting roles

A data-driven web application built using **Python, Streamlit, and SQL** to analyze invoice and payment data, track delays, and generate actionable business insights.

---

## 🚀 Features

* 📈 KPI Tracking (Total Revenue, Avg Delay, Invoice Count)
* 🗄️ SQL Integration (SQLite for data querying)
* ⏳ Late Payment Analysis
* ⚠️ Risk Classification (High Risk vs Low Risk customers)
* 🤖 ML-Based Prediction (Predict delayed payments)
* 🎛️ Interactive Filters (Customer-wise analysis)
* 📊 Dynamic Visualizations (Bar charts, Pie charts)

---

## 🛠️ Tech Stack

* Python
* Pandas
* Streamlit
* Plotly
* SQLite
* Scikit-learn

---

## 📈 Business Use Case

This project simulates a real-world **Order-to-Cash (O2C)** process used by companies to:

* Monitor cash flow
* Identify delayed payments
* Detect high-risk customers
* Optimize payment cycles
* Improve financial decision-making

---

## ▶️ How to Run Locally

```bash
py -m pip install -r requirements.txt
py -m streamlit run app.py
```
